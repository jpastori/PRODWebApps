Procedure 3: Apache Web Server Software Startup
(S50apache start).


      The Apache Web Server software startup procedure is done automatically when the LIBINT.SDSU.EDU server is started. It can also be done manually from the command line in a secure shell (ssh) session:
    1. Start the secure shell (ssh) software to create a telnet style session.

    2. Login as user oracle. (See Barry, Carol, John or Ron for current user account passwords.)

    3. Change to user root with the su command: su [return], Enter root password [return].

    4. Change to the /etc/rc3.d directory: cd /etc/rc3.d [return].

    5. Run the Apache Web Server software startup script S50apache: ./S50apache start [return].

    6. Check the status of the startup procedure by using the psgrep command: psgrep apache [return].

    7. Log out of the secure shell (ssh) software by typing: exit [return] and closing the window.

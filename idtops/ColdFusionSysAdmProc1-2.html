These procedures apply to both the LIBINT.SDSU.EDU and LIBWEB.SDSU.EDU servers.

Procedure 1: ColdFusion Server Software Startup
(jrun -start cfusion).


      The ColdFusion server software startup procedure is done automatically when the LIBINT.SDSU.EDU and LIBWEB.SDSU.EDU servers are started. It can also be done manually from the command line in a secure shell (ssh) session:
    1. Start the secure shell (ssh) software to create a telnet style session.

    2. Login as user oracle. (See Barry, Carol, John or Ron for current user account passwords.)

    3. Change to user root with the su command: su [return], Enter root password [return].

    4. Change to the /etc/init.d directory: cd /etc/init.d [return].

    5. List the rc.local file to copy Coldfusion MX 7 startup command: cat rc.local [return].

    6. Copy and paste the ColdFusion MX 7 Server software startup command from from under the 'start' label in the rc.local file to the command prompt:
      >/opt/coldfusionmx7/bin/jrun -start cfusion & [return].
      Note: the ampersand at the end of the line is necessary so that the ColdFusion MX 7 Server software starts as a background process.

    7. Check the status of the startup procedure by using the psgrep command: psgrep cfusion [return].

    8. Log out of the secure shell (ssh) software by typing: exit [return] and closing the window.



Procedure 2: ColdFusion Server Software Shutdown
(jrun -stop cfusion).


      The ColdFusion server software shutdown procedure is done automatically when the LIBINT.SDSU.EDU and LIBWEB.SDSU.EDU servers are shutdown. It can also be done manually from the command line in a secure shell (ssh) session
    1. Start the secure shell (ssh) software to create a telnet style session.

    2. Login as user oracle. (See Barry, Carol, John or Ron for current user account passwords.)

    3. Change to user root with the su command: su [return], Enter root password [return].

    4. Change to the /etc/init.d directory: cd /etc/init.d [return].

    5. List the rc.local file to copy Coldfusion MX 7 startup command: cat rc.local [return].

    6. Copy and paste the ColdFusion MX 7 Server software shutdown command from under the 'stop' label in the rc.local file to the command prompt:
      >/opt/coldfusionmx7/bin/jrun -stop cfusion [return]. The ampersand is not required for this command.

    7. Check the status of the shutdown procedure by using the psgrep command: psgrep cfusion [return].

    8. Log out of the secure shell (ssh) software by typing: exit [return] and closing the window.
